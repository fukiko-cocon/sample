# デザイン画面のプレビュー

design.pen の 01 / 02 画面をブラウザで見るための HTML です。

## 重要: 開発サーバーはこのプロジェクト（2026test）から起動してください

**必ずプロジェクトルート `2026test` で `npm run dev` を実行してください。**

別のディレクトリ（例: rcos など）で起動していると、この public フォルダの HTML が読まれず、別アプリの画面が表示されます。

```bash
cd /Users/inouefukiko/AI_Mockup/2026test
npm run dev
```

起動後、表示された URL（例: http://localhost:5175/）のホスト:ポートを使って以下を開きます。

- 01 画面: http://localhost:5175/design-01.html
- 02 画面: http://localhost:5175/design-02.html
- 02 画面（別URL）: http://localhost:5175/design-02-screen.html
