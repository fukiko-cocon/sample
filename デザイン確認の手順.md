# デザイン画面（01・02）をブラウザで確認する手順

## いま起きていること

**localhost:5175 では、別のプロジェクト（rcos など）のサーバーが動いている可能性があります。**

そのため、`design-02.html` や `page-02.html` を開いても、**このプロジェクト（2026test）の HTML ではなく、別アプリのログイン画面（メールアドレス・名前・送信）** が表示されています。

## 正しく表示するための手順

### 1. いま動いている開発サーバーを止める

- localhost:5175 で動かしているターミナルで **Ctrl+C** を押してサーバーを停止する
- ほかにも `npm run dev` や `vite` を実行しているターミナルがあれば、すべて止める

### 2. このプロジェクト（2026test）だけから起動する

**Cursor のターミナルで、必ずこのプロジェクトのフォルダに移動してから起動してください。**

```bash
cd /Users/inouefukiko/AI_Mockup/2026test
npm run dev:design
```

（`npm run dev` でも動きますが、`dev:design` は「このプロジェクトから起動した」とメッセージが出ます。）

### 3. 表示された URL を使う

起動後、ターミナルに表示される URL を確認してください。

- 例: `http://localhost:5176/` や `http://localhost:5173/`
- **この表示されたポート番号** を使って、次のように開きます。

  - 01 画面: **http://localhost:【表示されたポート】/design-01.html**
  - 02 画面: **http://localhost:【表示されたポート】/page-02.html**

### 4. 正しく表示できているか確認する

- **01** … 白背景で「ログイン」・Select・Input・Button・青い円
- **02** … **グレー背景**（#e6e6e6）で、右上に **「02 (design.pen)」** の青いバッジが出ていれば、このプロジェクトの 02 が表示されています。

まだ「メールアドレス」「名前」「送信」の白い画面だけの場合は、**別プロジェクトのサーバー** を見ているので、もう一度「1. サーバーを止める」からやり直してください。
